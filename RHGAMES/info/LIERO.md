# Liero
*(page., 1999)*

Tato hra stále čeká na někoho,
kdo jí napíše hezký popis.
Prozatím je zde pár odstavců z
MobyGames.

This game is still waiting for someone
to write a nice description. For now,
here's few paragraphs from MobyGames:

Liero is like Worms but with players
battling in realtime instead of
turn-based. The game combines the
gameplay of MoleZ with the splitscreen
mode of Tunneler.

At the start of the game, each player
chooses 5 weapons from a vast weapons
array, 40 in total. The weapons range
from simple pistols and shotguns to
bazookas and the ever deadly cluster
bombs (known in the game as the
chiquita bomb)

https://www.mobygames.com/game/2411/liero/
